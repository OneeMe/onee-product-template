# Quality rubric v1

Score the response from 0.0 to 1.0 using only the user input and response.

- **1.0 — Strong pass:** Gives exactly three understandable workflow steps, identifies a concrete limitation, stays concise, and makes no unsupported product claims.
- **0.75 — Pass:** Covers the workflow and a limitation accurately, with only a minor clarity or formatting issue.
- **0.5 — Fail:** Partially answers the request but omits a required element, is materially vague, or includes an unsupported claim.
- **0.25 — Fail:** Mostly irrelevant, confusing, or substantially inaccurate.
- **0.0 — Fail:** Empty, unsafe, or unrelated.

The minimum passing score is 0.75. If evidence is insufficient to verify a product claim, treat that claim as unsupported rather than assuming it is true.
